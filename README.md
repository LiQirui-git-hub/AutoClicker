# AutoClicker
一个可以通过长按鼠标实现连续点击，并能智能区分拖动事件的连点器。

A clicker that allows continuous clicking by long-pressing the mouse and can intelligently distinguish drag events
## Quick Start
> [!NOTE]  
> 只支持Windows,目前已知兼容的有Windows 11 x64。其余版本可能会出现Bug
> 
> Only supports Windows. Currently known compatible version is Windows 11 x64. Other versions may have bugs.

```bash
pip install -r requirements.txt
python main.py
```
